# Slack AI Agent 🤖

An AI-powered Slack bot that analyzes Slack workspace members, generates insights using LLMs, and sends intelligent reports directly to Slack channels.

## 🚀 Live Deployment

**Render URL:**  
https://slack-ai-agent-92xk.onrender.com

---

## 📌 Features

- Slack Bot integration using Slack Bolt
- AI-powered member analysis
- Generates intelligent summaries and insights
- PostgreSQL database integration
- Stores analysis results for tracking
- Sends automated reports to Slack channels
- REST API endpoints for triggering analysis
- Cloud deployed on Render

---

## 🛠️ Tech Stack

### Backend
- Node.js
- Express.js

### AI / LLM
- LangChain
- Groq LLM

### Database
- PostgreSQL

### Slack Integration
- Slack Bolt SDK
- Slack Web API

### Deployment
- Render
